TypeScript

TypeScriptとは？

TypeScript（タイプスクリプト）は、Microsoftによって開発されたJavaScriptのスーパーセットです。静的型付けシステムを追加することで、より安全で保守性の高いコードを書くことができます。

1. 基本的な型

TypeScriptの基本的な型システムを学びます。

// 基本的な型
let age: number = 25;
let name: string = "田中太郎";
let isStudent: boolean = true;
let colors: string[] = ["赤", "青", "緑"];

console.log("年齢:", age, "名前:", name);
console.log("学生:", isStudent, "色:", colors);

2. オブジェクト型

オブジェクトの型定義方法を学びます。

// オブジェクト型
let person: { name: string; age: number } = {
    name: "佐藤花子", age: 30
};

// オプショナルプロパティ（?）
let user: { name: string; email?: string } = {
    name: "山田太郎"
};
user.email = "yamada@example.com";

console.log("人物:", person, "ユーザー:", user);

3. 関数の型

関数の引数と戻り値に型を指定する方法を学びます。

// 関数の型注釈
function add(x: number, y: number): number {
    return x + y;
}

// オプション引数（?）
function greet(name: string, title?: string): string {
    return title ? `こんにちは、${title}${name}さん！` : `こんにちは、${name}さん！`;
}

console.log("5 + 3 =", add(5, 3));
console.log(greet("田中"), greet("佐藤", "さん"));

4. Union型

複数の型を組み合わせるUnion型を学びます。

// Union型（複数の型のいずれか）
let id: number | string;
id = 123;
id = "USER001";

// 型ガードで安全に処理
function formatId(id: number | string): string {
    return typeof id === "number" ? `ID: ${id.toString().padStart(6, "0")}` : `ID: ${id.toUpperCase()}`;
}

console.log(formatId(42), formatId("abc123"));

5. 型エイリアス

型に名前を付けて再利用する方法を学びます。

// 型エイリアス（再利用可能な型定義）
type Status = "pending" | "completed" | "failed";
type User = { id: number; name: string; status: Status };

let currentUser: User = { id: 1, name: "鈴木一郎", status: "pending" };

// 型安全な関数
function updateStatus(user: User, newStatus: Status): User {
    return { ...user, status: newStatus };
}

console.log("元:", currentUser);
console.log("更新後:", updateStatus(currentUser, "completed"));

6. 配列とタプル

配列とタプル（固定長配列）の型定義を学びます。

// 配列型
let numbers: number[] = [1, 2, 3];

// タプル型（固定長・型が決まった配列）
let coordinate: [number, number] = [10, 20];
let person: [string, number] = ["田中太郎", 25];

// 型推論
let fruits = ["りんご", "バナナ"];  // string[]として推論

console.log("数値配列:", numbers);
console.log("座標:", coordinate, "人物:", person);
console.log("果物:", fruits);

7. never, any, unknown型

特殊な型について学びます。

// any型（型チェック無効）
let anyValue: any = 42;
anyValue = "文字列";

// unknown型（安全なany）
let unknownValue: unknown = "何かの値";
if (typeof unknownValue === "string") {
    console.log("文字列として処理:", unknownValue.toUpperCase());
}

// never型（絶対に戻らない関数）
function processValue(value: string | number): string {
    if (typeof value === "string") return value.toUpperCase();
    if (typeof value === "number") return value.toString();
    throw new Error("予期しない型");  // never型
}

console.log("any:", anyValue, "処理結果:", processValue("hello"));

8. 型注釈 vs 型推論

明示的な型注釈とTypeScriptの自動型推論を学びます。

// 型推論（TypeScriptが自動で型を決定）
let inferredString = "これは文字列";  // string型として推論
let inferredNumber = 42;             // number型として推論

// 明示的な型注釈
let explicitString: string = "明示的に指定";
let explicitArray: string[] = ["a", "b"];

// オブジェクトの型推論
const user = { name: "佐藤", age: 25, isActive: true };

console.log("推論:", inferredString, inferredNumber);
console.log("明示:", explicitString, explicitArray);
console.log("オブジェクト推論:", user);
